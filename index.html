구글 시트 가져오기
Formly에서 내보낸 구글 시트의 URL을 붙여넣으세요.
시트의 상품들이 현재 목록에 추가됩니다.
OAuth Client ID가 아직 설정되지 않았어요. 아래에 입력해 주세요.
구글 시트 OAuth 연동
1단계. Google Cloud Console 접속 → 프로젝트 선택 또는 새로 만들기
2단계. API 및 서비스 > 라이브러리 → Google Sheets API 활성화
3단계. API 및 서비스 > 사용자 인증 정보 > 사용자 인증 정보 만들기 > OAuth 2.0 클라이언트 ID
4단계. 애플리케이션 유형: 웹 애플리케이션 → 승인된 JavaScript 원본에 현재 주소 추가
※ file:// 로 열면 작동 안 함. python3 -m http.server 8080 실행 후 http://localhost:8080으로 접속하세요.
